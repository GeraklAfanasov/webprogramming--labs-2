Пример использования фильтров в Jinja

Исходная фраза:

{{ phrase }}

Фильтр title:

{{ phrase | title }}

Фильтр striptags:

{{ phrase | striptags }}

Фильтр safe:

{{ phrase | safe }}

Фильтр replace:

{{ phrase | replace("сколько", "много") | replace("нам", "тут") }}

Фильтр upper:

{{ phrase | upper }}

Фильтр striptags + reverse:

{{ phrase | striptags | reverse }}
